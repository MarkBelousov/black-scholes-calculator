# Black-Scholes Calculator

A premium web-based Black-Scholes option pricing calculator with a stunning Black/Gold/Platinum theme.

## Features

- **Real-time Option Pricing** - Calculate European call and put option prices
- **Complete Greeks** - Delta, Gamma, Theta, Vega, and Rho
- **Implied Volatility Calculator** - Work backwards from market price to find IV
- **Interactive Charts** - Visualize how prices and Greeks change with underlying price
- **In-Depth Help Guide** - Learn everything about the Black-Scholes model
- **Stunning UI** - Black/Gold/Platinum luxury theme with glassmorphism effects

## Technologies Used

- Pure HTML/CSS/JavaScript (no backend required)
- Plotly.js for interactive charts
- Responsive design with mobile support
- Client-side Black-Scholes calculations

## Usage

Visit the live site: [Black-Scholes Calculator](https://YOUR_USERNAME.github.io/black-scholes-calculator/)

Or run locally:
1. Clone the repository
2. Open `docs/index.html` in your browser
3. No server or dependencies required!

## Input Parameters

- **S** - Spot Price (current price of underlying asset)
- **K** - Strike Price (exercise price)
- **T** - Time to Expiration (in years)
- **r** - Risk-Free Rate (annualized, enter as percentage)
- **σ** - Volatility (annualized standard deviation, enter as percentage)

## Deployment

This site is configured for GitHub Pages from the `docs/` folder.
